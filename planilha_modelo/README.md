# Planilha modelo

Este projeto usa uma planilha do Google Sheets para armazenar os dados enviados pelo ESP32 e organizar as informações usadas no dashboard.

A planilha modelo contém:

- aba `dados`;
- aba `resumo`;
- aba `dashboard`;
- aba `Diagnóstico`;
- aba `diagnosticoIA`;
- script do Google Apps Script associado à planilha.

## Como usar

1. Clique no link abaixo.
2. Faça uma cópia da planilha.
3. Use a cópia na sua própria conta Google.
4. Configure sua própria chave Gemini no Apps Script.
5. Crie sua própria implantação como Web App.

## Link da planilha modelo

[Copiar planilha modelo](https://docs.google.com/spreadsheets/d/1QTHLuNLvVn8JgMqz4i_d4QXfSozsNgKqgmKVz3vZsk8/edit?usp=copy)

## Importante

A planilha modelo não contém:

- chave da API Gemini;
- URL de Web App ativa;
- dados reais;
- acionadores automáticos.

Cada usuário deve configurar suas próprias credenciais.
